# CommandCabin

A cross-platform desktop productivity launcher inspired by the command-palette workflow: global hotkey, fast local search, command execution, and a plugin-first architecture.

## Product Direction

The first version focuses on a Windows-first MVP:

- Global hotkey launcher.
- Application, favorite file, folder, and URL search.
- Unified command registry and command execution.
- Local plugin runtime.
- Built-in plugins for calculator, clipboard history, text tools, and quick open.
- Local settings and history storage.
